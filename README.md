# 🎮 B3 Desktop - Application E-Sport

## 👥 Membres du Projet

- **Diop**
- **AKTAS Semih**
- **AHMED KAFFA**

## 📋 Description

Application desktop de gestion e-sport développée avec Electron+ react + tailwind, permettant aux joueurs de suivre leurs performances, participer à des compétitions et accéder à une boutique de produits gaming.

## ✨ Fonctionnalités

### 🏠 Accueil
- **Tableau de bord personnalisé** avec message de bienvenue
- **Section hero** mettant en avant les compétitions majeures (ex: VALORANT Champions Tour)
- **Lecture vidéo** des grands événements e-sport

### 📊 Performances
- **Suivi du rang** actuel (système de classement Radiant)
- **Statistiques KDA** (Kill/Death/Assist ratio) avec tendances
- **Taux de victoire** avec visualisation graphique
- **Historique des performances** et progression

### 🏆 Compétitions
- **Liste des tournois à venir** ( VALORANT)
- **Inscription rapide** aux compétitions
- **Compteur en temps réel** avant le début des événements
- **Catégorisation** par type de jeu et niveau

### 🛒 Boutique
- **Catalogue de produits gaming** (équipements, accessoires)
- **Promotions et réductions** mises en avant
- **Panier d'achat** intégré
- **Prix affichés en euros**

### 🧭 Navigation
- **Barre de navigation inférieure** avec 5 sections :
  - 🏠 Accueil
  - 📊 Statistiques
  - 👥 Équipes
  - 🎲 tournois
  - 👤 Profil utilisateur

### 🔔 Fonctionnalités Additionnelles
- **Notifications** en temps réel
- **Paramètres** personnalisables
- **Interface responsive** et moderne

## 🛠️ Technologies Utilisées

### Core
- **Electron** - Framework pour applications desktop multi-plateformes
- **React** - Bibliothèque JavaScript pour l'interface utilisateur
- **TypeScript** - Superset typé de JavaScript

## 🚀 Installation et Démarrage

### Prérequis
- Node.js (version recommandée : LTS)
- npm ou yarn

### Installation des dépendances
```bash
npm install
```

### Lancer l'application en mode développement
```bash
npm start
```
## 🎨 Design

- **Palette de couleurs** : Dark theme avec des tons bleus (#0a1628, #0d1b2e, #1a2942)
- **Typographie** : Police système avec hiérarchie claire
- **Icônes** : Emojis pour une interface moderne et accessible
- **Layout** : Design mobile-first avec navigation en bas d'écran
